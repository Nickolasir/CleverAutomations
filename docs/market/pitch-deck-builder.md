## Slide 9: Installation Details

### For Your Construction Team

**What gets installed during rough-in (Phase 1):**
- One Cat6 ethernet run from structured wiring panel to Clever Hub location
- Standard 120V outlet at hub location (same as alarm panel requirement)
- Standard device rough-ins: thermostat wire (already standard), smart lock prep (already standard for electronic deadbolts)

**What gets installed during trim-out (Phase 2):**
- Mount Clever Hub (wall bracket, 4 screws)
- Connect Cat6 + power
- Install smart thermostat (standard thermostat install)
- Install smart lock(s) (standard deadbolt install)
- Install any smart switches (standard switch install)
- Power on hub — auto-discovery runs, devices pair automatically

**Total added installation time**: 30 minutes beyond standard trim-out

**No specialized subcontractor needed.** Your electrician and door hardware installer already do this work. We provide a 5-page installation guide and phone support.

**QA Check**: After install, hub runs automated device check and reports status to builder dashboard. Green = all devices responding. Red = needs attention.
